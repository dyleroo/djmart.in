BLOGAMING

  • A Ruby on Rails application that presents a discussion board where users can communicate asynchronously on a variety of topics.
  • The discussion threads are maintained in a SQLite database, with a web interface that allows users to contribute to an existing topic or to create a new topic for discussion.
  • Source Code

WEATHERWATCHER

  • WeatherWatcher is a python application that allows a weather presenter to choose locations, days and times from a stored list and then parses the 5-day forecast data retrieved from the OpenWeatherMap API to generate a series of images that can be used in a weather forecast TV broadcast.
  • Users can also add additional locations to those supported by providing the latitude and longitude of the location as well as a label for display on the map.
  • Location and day/time information should be stored in an SQLite database so that the same presentation can be easily generated again in the future.
  • Source Code
